 - Приложение копирует файл через буфер заданного размера с помощью двух потоков (чтения и записи).
 - Поток чтения вычитывает заданный файл и копирует данные в буфер. По заполнению буфера поток чтения
 не останавливается, но и не вычитывает данных из файла до тех пор пока в буфере не появится место.
 - Поток записи записывает выходной файл, выбирая данные из буфера. Если буфер пуст, то поток работает
 в "холостом режиме".
 - Пользовательский интерфейс предоставляет возможность:
    -- задать файл для работы
    -- задать размер буфера в байтах
    -- увидеть текущее заполнение буфера
    -- управлять работой потока чтения: приостановка работы (при которой чтение приостанавливается)/возобновление.
    -- управлять работой потока записи: приостановка работы (при которой запись приостанавливается)/возобновление.
    -- увидеть текущий режим работы потоков (работает/приостановлен)
